## Projetos em destaque

**Engenharia de Dados**
- [etl-ofertas](https://github.com/adaltonvieira/etl-ofertas) — Pipeline ETL, SQLite → PostgreSQL, API REST (FastAPI) e dashboard.
- [ofertas-db](https://github.com/adaltonvieira/ofertas-db) — Modelagem relacional multi-tabela com SQLAlchemy + Alembic (migrations).

**Inteligência Artificial**
- [organizador-ia](https://github.com/adaltonvieira/organizador-ia) — Análise de produtos com LLM (Groq) e saída estruturada em JSON.
- [detector-golpes](https://github.com/adaltonvieira/detector-golpes) — Detector de phishing: regras determinísticas + LLM (IA + Cibersegurança).

**Machine Learning**
- [deteccao-fraudes](https://github.com/adaltonvieira/deteccao-fraudes) — Detecção de anomalias em transações (Isolation Forest vs z-score), AUC ~0,95.
